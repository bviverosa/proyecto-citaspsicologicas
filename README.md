Crear una carpeta para alojar la rama del proyecto

Dentro de la carpeta, en la terminal, ejecutar el comando git clone https://github.com/bviverosa/proyecto-citaspsicologicas.git

Ejecutar el comando npm install

 Si la terminal lanza el error mencionando que no se pueden ejecutar scripts externos, se debe utilizar el siguiente comando en powershell

 Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
